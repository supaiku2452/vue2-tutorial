# Todo Application

Vue.jsを利用したTodoアプリケーションです。

## 前提条件

- Node.jsがインストールされていること。

## 起動方法

1. `npm install`でパッケージをインストールする。
2. `webpack`を実行する。
3. **todoapp.html**をブラウザで表示する。
